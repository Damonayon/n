# 🤖 Нейро-Новости Бот — v2.0

Полностью автоматический Telegram-бот для канала об ИИ.  
**Бесплатно. Без сервера. Работает 24/7. Посты с картинками.**

---

## Как это работает

```
Каждые 6 часов:
  RSS-ленты (7 источников)
       ↓
  Gemini пишет пост + придумывает описание картинки
       ↓
  Pollinations AI рисует картинку (бесплатно, без ключа!)
       ↓
  Тебе в Telegram приходит: картинка + пост + кнопки
       ↓
  Ты нажимаешь ✅ → пост с картинкой уходит в канал
  (в течение 10 минут)
```

---

## Что улучшено в v2.0

| Было | Стало |
|------|-------|
| Только текст | Картинка + текст в каждом посте |
| Падало без уведомления | Уведомляет тебя при любой ошибке |
| Нет retry | 3 попытки при сбое Gemini API |
| Лимит минут GitHub исчерпывался | Pip-кэш экономит ~80% минут |
| Конфликты при одновременном запуске | Concurrency-блокировка |
| Жёсткие версии библиотек | Гибкие версии, не устареют |

## УСТАНОВКА — 5 шагов (≈ 20 минут)

---

### ШАГ 1 — Создаём Telegram-бота

1. Открой Telegram, найди **@BotFather**
2. Напиши: `/newbot`
3. Придумай имя: например `Нейро Новости`
4. Придумай username: например `neuro_news_helper_bot`
5. Получишь токен вида `7123456789:AAHdqTcvCHhvGmz6TfbhZxPk5Qabc`
6. **Сохрани токен**

---

### ШАГ 2 — Узнаём свой Telegram ID

1. Найди в Telegram **@userinfobot**
2. Напиши `/start`
3. Получишь число вида `123456789`
4. **Сохрани это число**

---

### ШАГ 3 — Получаем Gemini API Key

1. Перейди: **https://aistudio.google.com**
2. Войди через Google-аккаунт
3. Левое меню → **Get API key** → **Create API key**
4. **Сохрани ключ** (длинная строка)

> Бесплатно: 1500 запросов в день — хватит на 50 постов

---

### ШАГ 4 — Создаём канал и добавляем бота

1. Создай канал в Telegram
2. Настройки канала → **Администраторы** → **Добавить**
3. Найди своего бота → дай право **Публикация сообщений**
4. Узнай ID канала:
   - Публичный канал: просто `@username_канала`
   - Приватный: напиши **@username_to_id_bot** → он покажет `-1001234567890`

---

### ШАГ 5 — GitHub (5 минут)

#### 5.1 Регистрация
Зайди на **https://github.com** → **Sign up** (бесплатно)

#### 5.2 Создаём репозиторий

> ⚠️ **ВАЖНО: выбери Public (публичный)**
> Это снимает лимит на количество автозапусков.
> Секреты (API-ключи) в публичном репо всё равно скрыты — они хранятся отдельно.

1. Нажми **+** (верхний правый угол) → **New repository**
2. Название: `neuro-news-bot`
3. Выбери **Public** ← обязательно!
4. Поставь галочку **Add a README file**
5. Нажми **Create repository**

#### 5.3 Загружаем файлы

1. На странице репозитория → **Add file** → **Upload files**
2. Загружаем всё содержимое архива:
   - папку `scripts/`
   - папку `.github/`
   - папку `data/`
   - файл `requirements.txt`
3. Нажми **Commit changes**

> ⚠️ Папка `.github` может быть скрыта:
> - **Mac**: Cmd+Shift+. — показать скрытые файлы
> - **Windows**: Проводник → Вид → Скрытые элементы ✓

#### 5.4 Добавляем секреты

1. Репозиторий → **Settings** → **Secrets and variables** → **Actions**
2. Нажимаем **New repository secret** 4 раза:

| Name | Value |
|------|-------|
| `TELEGRAM_BOT_TOKEN` | токен от BotFather |
| `TELEGRAM_MODERATOR_ID` | твой числовой ID |
| `TELEGRAM_CHANNEL_ID` | ID канала |
| `GEMINI_API_KEY` | ключ от Google AI Studio |

#### 5.5 Запускаем!

1. Репозиторий → вкладка **Actions**
2. Нажми **I understand my workflows, enable them**
3. Слева выбери **📝 Генерация поста**
4. Нажми **Run workflow** → **Run workflow**
5. Через 1-2 минуты в Telegram придёт первый пост с картинкой! 🎉

## Как пользоваться

Система работает **полностью автоматически**. Тебе только нажимать кнопки:

**✅ Опубликовать** — пост с картинкой уйдёт в канал в течение 10 минут  
**❌ Отклонить** — пост удаляется из очереди

## Расписание

| UTC | Москва | Действие |
|-----|--------|----------|
| 07:00 | 10:00 | Утренний пост |
| 13:00 | 16:00 | Дневной пост |

Хочешь изменить — открой `.github/workflows/generate.yml`, строка `cron`.

```
└── requirements.txt
```
